How To

FilaFind was created to aggregate materials for consumer 3D printers to help makers discover the materials needed for their next project.

The database responds immediately to changes in your selection to return up to date materials that could fulfill your project requirements.

Additionally, the database provides suggested processing parameters for a successful print using that particular material.

If you like the material properties you see, I've included a buy-now link to take you directly to the supplier's listing page for that specific material!
